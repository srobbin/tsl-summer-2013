Reset stylesheet

CSS resets include a handful of rule sets that take every common HTML element and scale them down to one unified style. These resets involve removing any sizing, margins, paddings, or additional styles. Resets need to be the very first CSS styles to be rendered to ensure that all the styles there after are being applied to the skeleton of a page. Eric Meyer's Reset Stylesheet

Typeface vs. Font

Typeface

A typeface is what you see. It is the artistic impression of how text looks, feels, and reads.

Font

A font is a file that includes a typeface. Using a font on a computer allows that computer to access the typeface.

Color

We can specify the color of text by using keywords like red, or rgb(255,0,0), or #FF0000 (hex values).

Font Properties

font-family

Specify a list of fallback font families.

font-size

Size in px, and other values.

Weights, like normal, bold, lighter, 100, 400, 700,

font-style

Styles, like italic, oblique

font-variant

Variants, like small-caps

Text properties

text-align

Do you want the text to align left, right, center or justified?

text-decoration

Decorate your text, like none, underline, overline, line-through, blink (not all browsers support)

text-indent

Put some horizontal space before the first line of text.

text-transform

uppercase, lowercase, capitalize
